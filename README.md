# tend

A calm, private companion for a 5-friend Artist's Way circle. Mobile-first PWA,
installable to the home screen.

Built from the Claude Design mockup (`tend.html`), in the warm Resurface style
(Fraunces + JetBrains Mono on parchment, plum accent).

## Five screens

- **Today** — Morning Pages (one-tap checkbox), Artist Date for the week, and the
  week's exercise checklist. Tapping a card's checkbox completes it; tapping the
  card body opens a private detail sheet to write a note or answer anytime.
- **You** — your journey: a 12-week mood arc, past Artist Dates, and your exercise
  answers (re-readable / editable).
- **Circle** — all 5 of you at a glance: each person's current week, pages, Artist
  Date, exercises, and mood. No likes, no comments.
- **Ideas** — the Artist Date library. Toggle between **Mine** (your private
  backlog) and **Shared** (the group's pool). Bookmark ideas to save them, check
  them off when you've done them, filter by category, and add your own with a
  "share with the circle" toggle. Tags: category · cost · indoor/outdoor ·
  solo/group.
- **Check-in** — filled in before the weekly call: mood picker + "looking forward
  to" + an optional "something to share."

## Run it

```bash
npm install
npm run dev      # local dev at http://localhost:5173
npm run build    # static output in dist/
npm run preview  # serve the production build
```

## Notes

- **Data is local for now.** Your progress persists in the browser via
  `localStorage` (key `tend.me.v2`); the four friends are mock data. The next
  phase swaps this for a real backend (magic-link login + a shared circle).
- **Installable:** open in Safari on iPhone → Share → Add to Home Screen. Runs
  full-screen with safe-area insets; manifest + service worker are generated by
  `vite-plugin-pwa`.
- **Accent** is locked to plum (`#8A5E7E`). To change it, edit `--rs-accent` in
  `src/styles/app.css`.

## Structure

```
src/
  App.jsx              app shell: app bar, tabs, detail sheet, persistence
  data/seed.js         mock data (replace with backend next)
  lib/theme.js         color + font constants
  styles/              design tokens + app shell CSS
  components/          primitives (Icon, Checkbox, Avatar, …), DetailSheet
  screens/             Today, Journey, Group, Ideas, Checkin
```
